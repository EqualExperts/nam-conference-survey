## Iteration Structure

Each iteration contains:
- `discovery/` - Interviews, observations, synthesis (see discovery/README.md)
- `stories/` - User stories extracted from synthesis
- `story-maps/` - Miro board sync metadata
- `design/` - Screenshots and design artifacts

## Naming Format
