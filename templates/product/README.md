# Templates - Product Management Output Structures

This directory contains templates that define the structure of generated artifacts for product management.

## Templates

Structure templates for PM artifacts:

| File | Used By | Purpose |
|------|---------|---------|
| story-template-llm-dev.md | /req | Story format for LLM-centric development |
| story-template-human-dev.md | /req | Story format for human-centric development |
| interview-template.md | /iter | Interview documentation structure |
| synthesis-template.md | /synth | Synthesis output format |
| release-template.md | /rel | Release documentation format |
| backlog-template.md | /req | Backlog structure |
| product-spec-template.md | /rel | Product specification format |
| meta-synthesis-template.md | - | Cross-iteration analysis format |
| rule-template.md | - | LLM prompting rule format |

**When to read**: When creating PM artifacts
**When to skip**: Engineering tasks

---

## Customization

Edit these templates to change the structure of generated outputs. Templates are referenced by prompts and slash commands.
